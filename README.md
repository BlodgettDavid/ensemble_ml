# ensemble_ml

This project showcases ensemble learning techniques—including bagging, boosting,
and extensions beyond classification and regression such as clustering and dimensionality reduction.

## Requirements
- Python 3.13 or later
- A virtual environment (recommended)

## Getting Started
After cloning this repo, run the appropriate setup script:

**Windows**:
`setup_env.bat`

**macOS/Linux**:
`./setup_env.sh`

Once setup is complete, start the project using:
`python mainworkflow.py`
